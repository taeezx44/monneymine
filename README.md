# MoneyMind — AI Smart Expense + Income Tracker

## Projects
- `moneymind-web/`      Next.js 15 Dashboard
- `moneymind-api/`      Node.js TypeScript REST API  
- `moneymind-mobile/`   React Native (Expo)
- `supabase/`           Schema SQL + Edge Functions
- `deploy/`             Railway / Vercel / EAS configs

## Supabase Project
- URL: https://mdhpjltygrhdpltioqfe.supabase.co
- Region: ap-southeast-1 (Singapore)

## Quick Start
1. รัน `supabase/schema.sql` ใน Supabase SQL Editor
2. ใส่ `SUPABASE_SERVICE_ROLE_KEY` ใน .env ทุกไฟล์
3. `cd moneymind-api && npm install && npm run dev`
4. `cd moneymind-web && npm install && npm run dev`
5. `cd moneymind-mobile && npm install && npx expo start`
